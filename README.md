# Gear Web P2P File Transferer

[Gear-P2P (hermygong.com)](https://p2pfile.hermygong.com/)

不经过第三方服务器，利用 WebRTC 提供加密文件传输功能。

## 使用

连接对方：输入对方名称，点击连接

聊天：连接后在左侧聊天栏进行

传输文件：点击对方头像，选择文件发送，传输进度会在聊天栏更新
